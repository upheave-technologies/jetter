# Roadmap

Human narrative only. The authoritative structure (which initiative is on which horizon, and its status) lives on each initiative file at `system/initiatives/INIT-*.md` and is computed into `system/state.json` by `.claude/scripts/captain-compile.mjs`. This document exists so a person can read the direction in prose. Mention each active and proposed initiative by `id` — the compiler flags any that drift out of this narrative.

## Now

What we are actively building and why. Reference initiatives by id, e.g. INIT-001.

## Next

What's queued behind the current work. INIT-00N.

## Later

Direction we've committed to but haven't scheduled. Deliberately vague is fine here.
